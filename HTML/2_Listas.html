Conhecendo as principais Listas

Existem 3 tipos principais de listas:

Lista não ordenada - ul

Gerealmente usamos a ul quando a ordem dos itens não importa.

  • Tópico 1
  • Tópico 2
  • Tópico 1
  • Tópico 2

Lista ordenada- ol

Utilizamos a ol quando precisamos respeitar a ordem dos itens.

  1. Abre
  2. Fecha

Lista ordenada estilizada - ol type="A","a","I","i"

ol type="" estiliza a lista ordenada.

Lista de definição - dl

Gerealmente usamos dl para definir termos e descrições.

HTML
Linguagem de marcação para criar páginas web.
CSS
Folas de estilo que define o design das páginas web
